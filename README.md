* Ogrodje programa v Javi


* PRINT metode
```java
System.out.print("Hello!"); // ne gre v nov odstavek

System.out.println("Hello!"); //gre v nov odstavek

System.out.printf("Hello!\n"); // \n, s tem določilom bomo skočili v novo vrstico
```
